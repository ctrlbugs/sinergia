
   ```bash
   npm install
   ```

3. **Start the Next.js development server**:
   ```bash
   npm run dev
   ```

4. **Open your browser** and go to:
   ```
   http://localhost:3002
   ```

**If you see port 8000 or http-server:**

- Make sure you're in the `landing-page-nextjs` folder
- Navigate to `landing-page-nextjs` folder first

**If port 3002 is already in use:**
- Stop the other application using port 3002
- Or use: `npm run dev:3000` to use port 3000 instead
